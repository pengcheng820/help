Docker镜像下载代理

由于国内纷纷关闭Docker镜像仓库，导致在国内无法下载Docker镜像
为了加速 Docker 镜像拉取，可以使用以下命令设置。

sudo tee /etc/docker/daemon.json <<EOF
{
    "registry-mirrors": ["https://docker.pengcheng.team"]
}
EOF
# 重载 Docker 配置
sudo systemctl  daemon-reload
# 重启 Docker 服务
sudo systemctl restart docker

NSFWJS接口

NSFWJS 是一个基于 TensorFlow.js 的 JavaScript 库，可以在浏览器或 Node.js 环境中检测不适宜的图像内容。

接口地址


https://nsfwjs.pengcheng.team/classify

请求示例

属性字段：image

POST /classify HTTP/1.1
Content-Type: multipart/form-data
